# Crosti

Статический просмотрщик схем вышивки для GitHub Pages и iPad Safari.

После публикации откройте сайт в Safari на iPad, нажмите Share и выберите Add to Home Screen.

## Android

Репозиторий содержит Android-обертку над тем же интерфейсом Crosti. APK собирается через GitHub Actions: откройте вкладку Actions, выберите последний запуск Android APK и скачайте artifact `crosti-debug-apk`.

Android-приложение сохраняет последний импортированный проект и рабочее состояние во внутренней памяти, поэтому после перезапуска JSON не нужно открывать заново.
